# haddock

Light theme based on [Haskell Haddock](https://haskell-haddock.readthedocs.io/)
documentation tool. Light gray background for visual separation.

## Colors

| Property         | Value     |
| ---------------- | --------- |
| Background color | `#f8f8f8` |
| Text color       | inherited |

## Usage

```yaml
# pandoc.yaml
code-theme: haddock
```
