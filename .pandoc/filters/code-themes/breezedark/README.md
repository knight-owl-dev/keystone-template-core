# breezedark

Dark theme based on the [KDE Breeze Dark](https://community.kde.org/Breeze)
color scheme. Cool blue-gray background with warm off-white text.

## Colors

| Property         | Value     |
| ---------------- | --------- |
| Background color | `#232629` |
| Text color       | `#cfcfc2` |

## Usage

```yaml
# pandoc.yaml
code-theme: breezedark
```
