# tango

Light theme with a warm gray background. Based on the
[Tango Desktop Project](http://tango-project.org/) color palette.

This is the default code theme — used when `code-theme` is not set in
`pandoc.yaml`.

## Colors

| Property         | Value     |
| ---------------- | --------- |
| Background color | `#f8f8f8` |
| Text color       | inherited |

## Usage

```yaml
# pandoc.yaml — default, no code-theme key needed
code-theme: tango
```
