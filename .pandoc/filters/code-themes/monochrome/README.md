# monochrome

Grayscale theme from [Pandoc](https://pandoc.org/MANUAL.html#syntax-highlighting)
that uses only bold and italic for emphasis — no syntax colors. Light gray
background for visual separation.

## Colors

| Property         | Value     |
| ---------------- | --------- |
| Background color | `#f8f8f8` |
| Text color       | inherited |

## Usage

```yaml
# pandoc.yaml
code-theme: monochrome
```
