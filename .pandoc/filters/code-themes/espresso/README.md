# espresso

Dark theme based on [ultraviolet](https://ultraviolet.textmate.org/)'s
espresso\_libre color scheme. Warm brown background with tan text.

## Colors

| Property         | Value     |
| ---------------- | --------- |
| Background color | `#2a211c` |
| Text color       | `#bdae9d` |

## Usage

```yaml
# pandoc.yaml
code-theme: espresso
```
