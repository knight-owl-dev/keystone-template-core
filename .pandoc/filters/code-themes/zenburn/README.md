# zenburn

Dark theme with low-contrast warm tones based on
[Zenburn](https://kippura.org/zenburnpage/), designed to reduce eye strain
during extended reading.

## Colors

| Property         | Value     |
| ---------------- | --------- |
| Background color | `#303030` |
| Text color       | `#cccccc` |

## Usage

```yaml
# pandoc.yaml
code-theme: zenburn
```
